# Identificação do Projeto de Redes
## Nome do Técnico responsável: Daniel Rodrigo Ferreira dos Santos

## Resumo descritivo

O objetivo do projeto é apresentar uma planta de laboratório de redes em um condomínio de forma simplificada e funcional, que permita a colaboração institucional e o partilhamento de recursos de rede, software e hardware em comunidade. O grupo de trabalho proposto para essa simulação é "Enfoque Intradisciplinar na Fonoaudiologia: Atençao a Saúde nas Várias Fases da Vida", do Centro de Ciências Biológicas e da Saúde, da Universidade Federal de Sergipe.
